# Navigatie

Eenmaal binnen op een server is alles hooguit drie tikken weg. Dit hoofdstuk laat zien waar alles zit.

## De drie tabbladen

Elke server heeft drie hoofdonderdelen:

- **Home** — recent afgespeelde items en uitgelicht materiaal uit je bibliotheken.
- **Bibliotheek** — alles op de server, te doorbladeren en te sorteren.
- **Instellingen** — je taal- en afspeelvoorkeuren voor deze server, plus serverinformatie.

Op telefoons staan deze in een balk onderaan het scherm. Op desktop, web en tv verschijnen dezelfde tabbladen als een navigatiebalk langs de linkerrand — dezelfde bestemmingen, alleen ingericht voor een groter scherm.

![Het startscherm met de navigatietabbladen](../images/nl/home-dashboard.png)

## Wisselen tussen servers

De app is gebouwd voor meer dan één server. Om te wisselen ga je vanaf het startscherm van de server terug naar het **serveroverzicht** — de lijst die je zag toen je de app voor het eerst opende. Kies een andere server en je bent binnen, ingelogd met het account van díe server.

Het afspelen loopt gewoon door terwijl je wisselt: je kunt op de ene server bladeren terwijl muziek van een andere blijft spelen, en de minispeler blijft overal zichtbaar.
