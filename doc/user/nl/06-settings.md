# Instellingen

Elke server heeft een eigen tabblad **Instellingen**. Je voorkeuren worden op de server bewaard, per gebruiker — stel ze één keer in en elk apparaat waarop je inlogt gebruikt ze.

![Het instellingenoverzicht](../images/nl/settings.png)

## Taalvoorkeuren

Kies je voorkeurstaal voor **gesproken audio** en voor **ondertiteling**. Biedt een film of aflevering meerdere audiosporen of ondertitels, dan kiest de speler automatisch jouw voorkeuren; per video kun je ze via de spelerknoppen alsnog aanpassen.

![De pagina met taalvoorkeuren](../images/nl/settings-languages.png)

## Afspeelinstellingen

Bepaal hoe media naar dit account wordt gestreamd:

- **Direct afspelen** — speel bestanden ongewijzigd af als je apparaat het formaat aankan, voor de beste kwaliteit en de minste serverbelasting.
- **Transcoderen** — laat de server media waar nodig ter plekke omzetten.
- **Maximale videohoogte** — begrens de resolutie (bijvoorbeeld op een trage of gebundelde verbinding).

Omdat deze instellingen op de server staan, reizen ze met je mee tussen apparaten.

![De pagina met afspeelinstellingen](../images/nl/settings-playback.png)

## Clusterinformatie

De clusterpagina toont de gezondheid van je Ister-server: op welke nodes hij draait en hoe die ervoor staan. Handig voor een snelle check of alles draait.

![De pagina met clusterinformatie](../images/nl/settings-cluster.png)

## Activiteit

De activiteitenpagina toont wat er recent op de server gebeurde — wat er is afgespeeld en wanneer.

![De activiteitenpagina van de server](../images/nl/activity.png)
