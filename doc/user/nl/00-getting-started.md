# Aan de slag

Ister is je eigen mediaserver, en met deze app kijk, luister en lees je eruit — op je telefoon, tv, desktop of in de browser. Dit hoofdstuk brengt je van een verse installatie naar je eerste startscherm.

## Een server toevoegen

Als je de app voor het eerst opent, kom je op het serveroverzicht. Tik op **Server toevoegen** en typ het adres van je Ister-server — bijvoorbeeld `media.voorbeeld.nl` of `192.168.1.10:8080/api`.

![Het serveroverzicht met het veld om een server toe te voegen](../images/nl/server-overview.png)

Meer dan het adres hoef je niet te weten: de app vraagt de server om zichzelf voor te stellen en ontdekt automatisch de naam en de manier van inloggen. Klopt het adres, dan verschijnt de naam van de server en kun je verder.

## Inloggen

Ister werkt met single sign-on: de app opent de vertrouwde inlogpagina van je server in een beveiligd venster. Log daar in — met hetzelfde account dat je ook elders gebruikt — en je komt automatisch terug in de app. Je sessie wordt per server bewaard, dus dit doe je maar één keer per server.

## Je startscherm

Na het inloggen kom je op het startscherm van de server: waar je onlangs naar keek of luisterde, plus rijen uit je bibliotheken.

![Het startscherm na het inloggen](../images/nl/home-dashboard.png)

Je kunt zo veel servers toevoegen als je wilt; elke server houdt zijn eigen login en instellingen. Hoe je ertussen wisselt lees je in [Navigatie](01-navigation.md). Wil je een server verwijderen, tik dan op het prullenbakpictogram ernaast op het overzicht — daarmee wordt ook de bewaarde login vergeten.
